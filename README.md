Um sistema web para gerenciamento de informações sobre carros. Os usuários podem visualizar, buscar e adicionar informações sobre carros, incluindo modelo, marca, valor, ano e outras características.

Recursos Principais:
- Criar usuário
- Login / Logout
- Visualizar uma lista de carros cadastrados
- Buscar carros por marca
- Adicionar, editar e excluir carros do banco de dados
- Dashboard com informações do estoque

Tecnologias Utilizadas:
- Django Framework
- HTML/CSS
- Power BI

Instruções de Uso:
1. Clone o repositório.
2. Instale as dependências usando pip.
3. Configure o ambiente e o banco de dados.
4. Inicie o servidor Django.
